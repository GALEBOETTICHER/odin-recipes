# odin-recipes

Hi, I am Antoine Desabille, an aspiring full-stack web developer. 

This is my **Recipes Project**. I built this practice website to demonstrate how to create a structured recipe site using core web development principles.

### Key Features
* **Page Navigation**: Links connecting the main page to individual recipe pages.
* **Media Integration**: Optimized images using various formats including SVG, GIF, PNG, and JPG.
* **Rich Content**: Additional media elements to enhance the visual appeal and user experience.
* **Structured Data**: Organized information using ordered and unordered HTML lists.
